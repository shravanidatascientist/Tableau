# Retail Sales Intelligence: Statistical Analysis of Store Performance

[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/)
[![Streamlit](https://img.shields.io/badge/Streamlit-App-red.svg)](https://streamlit.io/)

An interactive statistical analysis of retail sales data featuring hypothesis testing,
confidence intervals, and dynamic visualizations.

---

## Project Overview

This project analyzes sales performance across 4 product categories (Electronics, Clothing,
Home, Sports) to help retail managers make data-driven decisions.

**Key Question:** Does our average sales volume significantly differ from the target of 20 units?

---

## How to Run

### Interactive App (Streamlit)

```bash
streamlit run app.py
```
